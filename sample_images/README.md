# Sample Images Directory

This directory should contain sample crop disease images for testing.

## Recommended Images to Download:

### Tomato Diseases:
1. `tomato_early_blight.jpg` - Tomato leaf with early blight symptoms
2. `tomato_late_blight.jpg` - Tomato leaf with late blight
3. `tomato_healthy.jpg` - Healthy tomato plant

### Wheat Diseases:
4. `wheat_yellow_rust.jpg` - Wheat with yellow rust
5. `wheat_brown_rust.jpg` - Wheat with brown rust
6. `wheat_healthy.jpg` - Healthy wheat

## Where to Find Images:

You can download these images from:
- PlantVillage Dataset
- Agricultural research websites
- Your own field photos

## Image Guidelines:

- Clear, well-lit photos
- Focus on leaves showing symptoms
- JPG or PNG format
- At least 500x500 pixels
- File size under 5MB

## Note:

For the hackathon demo, use 5-10 "perfect" examples that clearly show disease symptoms or healthy plants. This ensures the demo works reliably.
